# Movie Library (Личная кинотека)

**Автор:** Искандер Айнуллов

## Описание
GUI-приложение для хранения информации о фильмах. Позволяет добавлять фильмы с названием, жанром, годом выпуска и рейтингом. Данные сохраняются в JSON-файл и загружаются при старте. Поддерживается фильтрация по жанру и году выпуска.

## Установка и запуск
1. Убедитесь, что установлен Python 3.6+.
2. Скачайте файл `movie_library.py`.
3. Запустите:
   ```bash
   python movie_library.py
